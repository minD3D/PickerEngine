아래에 제공한 인스타그램 계정 정보를 바탕으로 분석해줘.

목적:
- 인플루언서 협업 제안 메일 작성을 위한 계정 이해
- 계정 운영자가 “이 계정을 잘 관찰했다”고 느낄 수 있는 표현 추출

분석 방식:
- 제공된 텍스트와 이미지의 분위기, 반복되는 정서, 이모지, 주제어를 함께 분석
- 정보가 제한적인 경우에도 분위기와 패턴을 기반으로 부드럽게 추론
- 개인 기록 계정, 육아 계정, 소규모 계정도 분석 대상에 포함

요청 사항:
1. 계정의 특징을 나타내는 한국어 키워드 8~15개를 추출해줘
   - 1~3단어의 짧은 명사 또는 형용사
   - 과장된 마케팅 용어, 고유명사 제외
   - 감성, 태도, 기록 방식, 분위기 중심
   - 정보가 부족하더라도 결과를 비우지 말 것

2. 아래 기준으로 계정을 관찰 기반으로 분류해줘
   - 전반적인 분위기/정서
   - 추정 가능한 라이프스테이지 (범위형·중립적 표현 허용)
   - 콘텐츠 사용 목적 (기록 / 공유 / 정보 / 혼합 등)
   - 협업 접근성에 대한 관찰 (명확 / 보통 / 판단 어려움)

중요:
- 확신이 없는 경우 ‘판단 어려움’, ‘혼합형’, ‘중립적’ 표현을 사용할 것
- 평가하거나 단정하지 말고, 관찰 결과처럼 서술할 것
- 어떤 경우에도 빈 응답을 반환하지 말 것

출력 규칙:
- 반드시 JSON 형식으로만 출력
- 설명 문장, 주석, 추가 텍스트 없이 JSON만 반환
- 아래 구조를 정확히 따를 것

{
  "keywords": ["키워드1", "키워드2", "..."],
  "category": [
    "분위기/정서 관찰",
    "라이프스테이지 추정",
    "콘텐츠 목적",
    "협업 접근성 관찰"
  ]
}
